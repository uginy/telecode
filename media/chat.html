Core
Autonomous loop with plan → tools → apply → verify cycle for real coding tasks.
Remote
Telegram and WhatsApp channels with unified start/stop and access policies for safe remote control.
Tools
Workspace tools for reading/editing files, searching code, and running shell commands.
  • ◆ Autonomous agent loop: plans, executes tools, edits code, and iterates until done.
  • ◆ Telegram and WhatsApp remote control: run tasks from your phone while VS Code stays open.
  • ◆ Multi-provider LLM support: OpenAI, Anthropic, Gemini, Moonshot, OpenRouter, Ollama.
  • ◆ Native coding toolchain: read/write/edit, glob/grep, bash, workspace context.
