Built for real developers.

Autonomous Execution

The agent doesn't just respond. It plans multi-step tasks, reads your codebase, runs bash commands, and edits files in a continuous loop.

Telegram Remote Control

Send tasks from your phone, sofa, or even another country. Your IDE is the server. Telegram is the remote.

Any AI Provider

OpenRouter, OpenAI, Anthropic, Google Gemini, Ollama. One interface, any model. Even free ones via OpenRouter.

Native Toolchain

read_file, write_file, edit_file, glob, grep, bash — a full coding environment available to the agent.

Open Source

No black boxes. Inspect, fork, extend. TeleCode AI is built to be fully transparent and hackable.

Session Memory

The agent remembers context within a workspace session, making multi-step work smooth and coherent.
